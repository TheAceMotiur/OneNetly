## Description

[Sentora](http://sentora.org) is based on an original fork of ZPanelCP, Sentora is an open-source web hosting control panel written in PHP and is designed to work with Linux, UNIX and the BSDs.

## Supported Systems

Sentora 2.0.2 supports **Ubuntu 22.04 LTS** and other major Linux distributions. For a complete list of supported operating systems and system requirements, see [SUPPORTED_SYSTEMS.md](SUPPORTED_SYSTEMS.md).
