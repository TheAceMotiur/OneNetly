# OneNetly - Social Sharing Widget

Best free social sharing widget for modern websites. Privacy-first, lightweight (5KB), and zero dependencies.

## Features

- 🚀 **Lightning Fast** - Under 5KB minified and gzipped
- 📱 **Fully Responsive** - Works perfectly on all devices
- 🎨 **Highly Customizable** - Colors, position, and social networks
- 🔒 **Privacy-Focused** - No tracking or data collection
- ⚡ **Zero Dependencies** - Pure vanilla JavaScript
- 🌐 **Multiple Networks** - Facebook, Twitter, LinkedIn, WhatsApp, Telegram, Reddit, Pinterest, Email

## Quick Start

```bash
# Install dependencies
npm install

# Development server
npm run dev

# Build for production
npm run build

# Generate static site
npm run generate

# Deploy to Cloudflare Pages
npm run deploy
```

## Tech Stack

- **Framework**: Nuxt.js 4
- **Styling**: Tailwind CSS 3
- **Deployment**: Cloudflare Pages
- **AdSense**: Google AdSense integration

## Project Structure

```
oneweb/
├── assets/         # Global CSS and assets
├── components/     # Vue components
├── layouts/        # Layout components
├── pages/          # Route pages
├── public/         # Static files
├── utils/          # Utility functions
└── nuxt.config.ts  # Nuxt configuration
```
