OneNetly Share Widget - Implementation Demo

This page demonstrates how to implement the OneNetly Social Share Widget on your website. The floating share buttons should appear on the left side of this page.

Note: This demo loads the widget from a local file. In production, you would host the share-widget.js file on your server or CDN.

Basic Implementation

Add this code to your website's HTML:

<!-- OneNetly Social Share Widget --> <script src="https://yourserver.com/share-widget.js"></script> <script> OneNetlyShare.init({ position: 'left', // 'left' or 'right' shape: 'circle', // 'circle' or 'square' size: '50px', // Button size animation: 'scale', // 'scale', 'bounce', or 'rotate' networks: ['facebook', 'twitter', 'linkedin', 'pinterest', 'whatsapp', 'telegram'] }); </script>

Implementation Examples

Example 1: Auto-initialization with data attributes

<script src="https://yourserver.com/share-widget.js" data-position="right" data-shape="square" data-size="60px" data-animation="bounce" data-networks='["facebook","twitter","linkedin","email"]'> </script>

Example 2: Custom configuration

<script src="https://yourserver.com/share-widget.js"></script> <script> OneNetlyShare.init({ position: 'right', shape: 'square', size: '45px', animation: 'rotate', networks: ['facebook', 'twitter', 'reddit', 'email'], customCSS: { facebook: '#4267B2', // Custom Facebook color twitter: '#1DA1F2', // Custom Twitter color reddit: '#FF4500', // Custom Reddit color email: '#34495e' // Custom Email color }, showMoreButton: true, moreNetworks: ['linkedin', 'pinterest', 'whatsapp', 'telegram', 'copy', 'print'] }); </script>

Example 3: Minimal setup

<script src="https://yourserver.com/share-widget.js"></script> <script> OneNetlyShare.init(); // Uses default settings </script>

Available Configuration Options

  • position: 'left' or 'right' (default: 'left')
  • shape: 'circle' or 'square' (default: 'circle')
  • size: Button size in pixels (default: '50px')
  • animation: 'scale', 'bounce', or 'rotate' (default: 'scale')
  • networks: Array of social networks to display (max 6)
  • customCSS: Object with custom colors for each network
  • showMoreButton: Boolean to show/hide the "More" button (default: true)
  • moreNetworks: Array of networks shown in the modal

Available Social Networks

facebook, twitter, linkedin, pinterest, whatsapp, telegram, reddit, tumblr, vk, email, copy, print
